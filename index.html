ferencfarkas.org

Official website of Ferenc Farkas: a famous hungarian classical music composer.

Built with Hugo static website generator, Hyperapp tiny reactive frontend framework and hosted on Netlify, this project illustrates what the JAMStack is capable of: speed, performance, security and cost efficiency.

It's also the first project which will use Ourcecode for content editing and demonstrate how non-tech people can maintain YAML and Markdwon data, version with Git and publish their website on a serverless infrastructure without any technological knowledge.

Ourcecode

Ever tried to have website authors working with Git and an IDE ?

CMS are perfect for those kind of users but they lock your projects into specific technologies and formats, especially regarding content..

Wouldn't it be so cool to have your authors using a version of Visual Studio Code specifically tailored to their needs ?

Well, that's what Ourcecode is trying to achieve: having website authors using technology-agnostic content formats (YAML, JSON, Markdown), Git versioning and branching system, making it easier for every one involved in web publishing process to work together.

It's highly inspired by Visual Studio Code and currently focusing on being a frontend for the Hugo static website generator. Once this is done and the API is fully in place, it could be used as a generic frontend for any web project.
